# LoadingSpinner Component

A loading spinner represents an unspecified loading time.

**Import:** `import { LoadingSpinner } from "@faire/slate/components/loading-spinner"`

## Props

| Prop | Type | Default | Required | Description |
|------|------|---------|----------|-------------|
| size | "xSmall" \| "small" \| "medium" \| "large" \| "xLarge" | "small" | No | Spinner size |
| color | string | slate-icon-default | No | Spinner color |

## Variants

| Size | Dimensions |
|------|------------|
| xSmall | 12px |
| small | 16px |
| medium | 24px |
| large | 32px |
| xLarge | 48px |
